Q11 to Q13 are subjective answer type questions, Answer them briefly.
11. What is the main difference between a web scraper and a web crawler?
Ans: A web scraper is used to programmatically analyse a web page(s) and download required information from them. Eg. Extract stock market data
A web crawler reads/browses (known as crawling) the web, usually for the purpose of indexing
Eg. What google might be doing so that we can get various links for a search results

12. What is ‘robots.txt’ file? What is the use of ‘robots.txt’ file?
Ans: ‘robots.txt’ is a text file present for every website hosted on the web. It is also knows as robots exclusion protocol or standards. It tells the web crawlers which pages of the website to crawl and index and which once not to crawl.
Uses: 
* Tells crawlers which pages to crawl and which once not to.
* Helps a website owner to give access to only certain crawlers and save bandwidth by excluding spam requests
* Helps website owner get better ranking with search engine optimization like allowing access to only those pages which have relevant and informative content and blocking certain pages from being crawled.

13. What are static and dynamic web pages?
Ans: Static web pages are those which remain unchanged or constant whenever you load, while dynamic web pages have changing content. Their content changes on the fly based whenever they are loaded. 
Eg. Static Page- About me page of a blog
Dynamic page – e-commerce websites – say the prices vary from time to time.
